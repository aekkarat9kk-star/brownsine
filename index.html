About Me

ผมเป็นนักพัฒนาเว็บไซต์ที่มีความหลงใหลในการสร้างสรรค์ประสบการณ์ดิจิทัลที่สวยงามและใช้งานง่าย มีความถนัดในการใช้เทคโนโลยีสมัยใหม่เพื่อเปลี่ยนไอเดียให้กลายเป็นเว็บไซต์ที่มีชีวิตชีวา

ผมเชื่อว่าเว็บไซต์ที่ดีไม่ได้มีแค่โค้ดที่สะอาด แต่ต้องตอบโจทย์ผู้ใช้และสร้างความประทับใจได้ด้วย ทักษะของผม: HTML5, CSS3, JavaScript (ES6+), React, Figma และ Adobe XD

My Portfolio

เว็บไซต์ Landing Page บริษัท

ออกแบบและพัฒนาหน้าเว็บไซต์หลักสำหรับบริษัท Tech Startup โดยเน้นดีไซน์ที่ดูทันสมัยและสะอาดตา

บล็อกส่วนตัว "Code & Coffee"

สร้างบล็อกส่วนตัวด้วยดีไซน์แบบมินิมอล เพื่อแชร์ความรู้ด้านการเขียนโค้ดและเรื่องราวกาแฟ

คอนเซ็ปต์หน้าสินค้า E-commerce

ออกแบบ UI/UX สำหรับหน้ารายละเอียดสินค้า โดยเน้นที่การใช้งานง่ายและกระตุ้นการตัดสินใจซื้อ

แอปพลิเคชันจัดการงาน (Task App)

พัฒนาแอปพลิเคชันบนมือถือสำหรับจัดการงานประจำวัน พร้อมฟีเจอร์แจ้งเตือนและซิงค์ข้อมูลระหว่างอุปกรณ์

เว็บไซต์ E-commerce สำหรับแบรนด์แฟชั่น

พัฒนาระบบร้านค้าออนไลน์เต็มรูปแบบ มีระบบจัดการสต็อกสินค้า, ตะกร้าสินค้า, และการชำระเงินออนไลน์

Portfolio Website สำหรับช่างภาพ

สร้างเว็บไซต์แสดงผลงานภาพถ่ายที่เน้นการแสดงภาพขนาดใหญ่และสวยงาม พร้อมระบบแกลเลอรีที่ใช้งานง่าย

เว็บไซต์ร้านอาหารพร้อมระบบจองโต๊ะ

ออกแบบเว็บไซต์สำหรับร้านอาหารอิตาเลียน แสดงเมนูอาหาร, แกลเลอรีภาพ และระบบจองโต๊ะออนไลน์

เว็บแอปพยากรณ์อากาศ (Weather App)

พัฒนาเว็บแอปสำหรับเช็คสภาพอากาศแบบเรียลไทม์ โดยดึงข้อมูลจาก API และแสดงผลในรูปแบบที่เข้าใจง่าย

แดชบอร์ดวิเคราะห์ข้อมูลการขาย

ออกแบบและสร้างแดชบอร์ดสำหรับแสดงผลข้อมูลการขายแบบเรียลไทม์ ช่วยให้ทีมผู้บริหารตัดสินใจได้รวดเร็วขึ้น

เว็บไซต์ฟิตเนสพร้อมระบบจองคลาส

พัฒนาเว็บไซต์สำหรับสตูดิโอโยคะและฟิตเนส พร้อมระบบให้สมาชิกสามารถจองคลาสออนไลน์ได้

เว็บไซต์สำหรับธุรกิจอสังหาริมทรัพย์

สร้างเว็บไซต์แสดงรายการบ้านและคอนโด พร้อมระบบค้นหาและฟิลเตอร์ที่มีประสิทธิภาพ

ออกแบบแบรนด์และโลโก้สำหรับคาเฟ่

สร้างอัตลักษณ์แบรนด์ (Branding) ให้กับร้านกาแฟเปิดใหม่ ตั้งแต่การออกแบบโลโก้ไปจนถึงสื่อส่งเสริมการขาย

ออกแบบ UI/UX แอปพลิเคชันทำสมาธิ

ออกแบบแอปพลิเคชันบนมือถือที่ช่วยในการทำสมาธิ เน้นประสบการณ์ที่สงบและใช้งานง่าย

เว็บไซต์สำหรับองค์กรไม่แสวงหาผลกำไร

ออกแบบเว็บไซต์เพื่อการกุศล เน้นการสื่อสารภารกิจและช่องทางการสนับสนุนที่เข้าถึงง่าย
